## 🎯 Objetivo
Aplicar metodologías ágiles (SCRUM) en un proyecto simulado de desarrollo de software, fomentando el pensamiento crítico y el trabajo colaborativo.

## 📖 Introducción
El proyecto consiste en el diseño de un sistema de cine con arquitectura web de tres capas, modelo de datos normalizado, diagramas UML y wireframes.  
Se utilizaron herramientas como Jira, Trello, Git y GitHub para la gestión y documentación.
